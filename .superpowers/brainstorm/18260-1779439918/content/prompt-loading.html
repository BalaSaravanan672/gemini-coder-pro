Section 2: Prompt Loading & Injection

How the Orchestrator initializes the persona

Initialization Sequence
  1. Load File: Read system-prompt.md from the config directory.
  2. Fallback: If missing, use a built-in "safe" default.
  3. Prefind History: Inject the prompt as the very first message in a new session.
  4. Zero-Turn Injection: Combine the prompt with the initial Context Map to ensure the AI has full context immediately.

Prompt Cache Optimization

By grouping the prompt and the file tree at the start, we improve response speed for all subsequent turns.
