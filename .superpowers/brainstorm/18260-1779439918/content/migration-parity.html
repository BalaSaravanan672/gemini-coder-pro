Section 2: Configuration & Tool Parity

Mapping existing features to the new SDK

Current (deprecated)
import { VertexAi } from '...';

const model = vertexAi.getGenerativeModel({
  model: 'gemini-2.0-flash-001'
});
New (Modern Vertex)
import { GoogleGenerativeAI } from '...';
import { GoogleAuth } from 'google-auth-library';

const auth = new GoogleAuth({ ... });
const client = await auth.getClient();
const token = (await client.getAccessToken()).access_token;

const genAI = new GoogleGenerativeAI(token);

Tool Definition Updates

The new SDK uses a toolConfig format that is slightly different from Vertex AI.

  • Parameter Schema: We will update the structure in orchestrator.ts to match the OpenAP v3 standard used by the new SDK.
  • Function Calling: The loop in processTurn will be surgically updated to handle the new response object shape.
