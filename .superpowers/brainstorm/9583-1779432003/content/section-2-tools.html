Tooling Logic

1

/diff

Uses `git diff` for uncommitted state and internal state for per-turn diffs.

2

/review & /security-review

Injects a "Specialist" system prompt (e.g., Security Auditor) into the history.

3

/simplify

A multi-step agentic process: Identify complexity -> Propose refactor -> Verify tests.
